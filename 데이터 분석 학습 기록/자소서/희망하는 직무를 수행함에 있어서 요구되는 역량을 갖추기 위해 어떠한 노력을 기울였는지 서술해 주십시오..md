
저는 3년간 통신 솔루션 회사에서 데이터 베이스 처리 , 통계 데이터 처리 프로세스를 개발 및 고도화 업무를 담당해왔습니다.
각 모듈간 TCP/UDP 통신을 통해서 호처리 모듈의 통계 데이터를 수집하고 ,가공된 데이터를 웹 GUI로 전달해주는 API 개발하며 데이터가 수집되는 시점 및 전달 과정을 이해 할 수 있었고, 
전처리 과정에 대한 이해를 높이기 위해 SQLD 자격증을 습득한 이력이 있습니다.
현재는 데이터 분석 및 엔지니어링 역량에 대한 강화를 위해 현재 웅진에서 제공해주는 교육을 듣고 프로젝트를 진행하고 있습니다. 
프로젝트를 진행하며 python , n8n을 활용해 데이터 파이프라인이 구축과 AI를 활용해서 파이프라인의 자동화 환경을 구성하는 법을 배우고 있으며,
사람의 손이 가지않더라도 경영인들이 한눈에 데이터를 볼수있는 환경 구축을 제공하는 것에 대한 시야를 넓히고 있습니다.
저의 역량으로 인해 회사의 인력 손실을 최소화하고 그와 동시에 빠른 인사이트를 도출하는 데이터 분석가 및 엔지니어가 되고싶습니다.

 [[1. 통계]] 
## **글 쓴 이유**

 - 데이터 분석을 위해서는 통계 관련 용어들에 대한 이해가 필요함을 느낌

 - 간단하게 자주 사용하는 용어들을 표로 정리해서 메모장 처럼 보기위해서 작성

## **통계 관련 언어 표**

|   |   |   |   |
|---|---|---|---|
|**구분**|**명칭 (한글/영문)**|**의미 (정의)**|**실무 예시**|
|**중심 경향**|**평균 (Mean)**|모든 관측값의 합을 개수로 나눈 값|우리 쇼핑몰 고객들의 평균 결제 금액|
|**중앙값 (Median)**|데이터를 크기 순으로 세웠을 때 정중앙에 위치한 값|연봉 데이터처럼 극단적인 값이 있을 때 실제 수준 파악|
|**최빈값 (Mode)**|데이터 중 가장 빈번하게 나타나는 값|이번 달 가장 많이 팔린 상품 카테고리|
|**산포도**|**분산 (Variance)**|데이터가 평균으로부터 얼마나 퍼져 있는지를 나타내는 척도|공장 기계 A와 B 중 생산 오차가 더 큰 기계 찾기|
|**표준편차  <br>(Standard Deviation)**|분산의 제곱근으로, 데이터의 원래 단위로 환산한 산포도|주식 수익률의 변동성(리스크) 측정|
|**데이터 위치**|**사분위수 (Quartiles)**|데이터를 4등분했을 때의 각 지점 (25%, 50%, 75%)|성적 상위 25% 이내의 우수 학생 선별|
|**이상치 (Outlier)**|다른 데이터와 현저하게 차이가 나는 아주 크거나 작은 값|센서 오류나 사기 결제(Abnormal) 데이터 탐지|
|**추론 통계**|**모집단 & 표본  <br>(Population & Sample)**|연구 대상 전체 집단과 그중 일부로 뽑은 집단|전 국민(모집단) 중 1,000명(표본)을 뽑아 진행하는 여론조사|
|**가설 검정  <br>(Hypothesis Test)**|표본을 통해 어떤 가설이 통계적으로 옳은지 판단하는 것|"기존 약보다 신약의 효과가 더 좋다"는 주장의 진위 확인|
|**P-값 (p-value)**|귀무가설이 맞다는 전제하에 현재 결과가 나올 확률|p-value가 0.05보다 작으면 "우연이 아니다(유의미하다)"라고 판단|
|**관계 분석**|**상관계수 (Correlation)**|두 변수 사이의 선형적 관계의 강도 (-1 ~ 1 사이)|광고비 지출과 매출액 사이의 연관성 정도 파악|
|**회귀분석  <br>(Regression)**|변수 간의 인과관계를 모델링하여 미래 값을 예측|지난 1년간의 기온 데이터를 바탕으로 내일 아이스크림 판매량 예측|

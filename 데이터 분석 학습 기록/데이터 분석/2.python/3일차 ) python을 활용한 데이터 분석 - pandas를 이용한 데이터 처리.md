 **- str 접근자를 사용해서 시리즈 또는 인덱스에 대해 문자열 조건 처리를 할수도 있다.**

|   |   |
|---|---|
|**구분**|**상세 내용**|
|**sr.str[index]**|sr의 각 원소(문자열)를 대괄호 안에 지정한 인덱스로 인덱싱합니다. 대괄호 안에 슬라이스를 지정하면 sr의 각 원소를 슬라이싱합니다.|
|**sr.str.split(pat)**|sr의 각 원소를 패턴으로 분리한 리스트를 시리즈로 반환합니다. expand 매개변수에 True를 지정하면 시리즈를 데이터프레임으로 변환합니다.|
|**sr.str.contains(pat)**|sr의 각 원소에서 패턴의 존재 여부를 불리언 시리즈로 반환합니다.|
|**sr.str.replace(pat, repl)**|sr의 각 원소에서 패턴에 맞는 문자열을 repl에 지정한 문자열로 변경합니다. 정규 표현식일 때 regex 매개변수에 True를 지정합니다. (기본값: False)|
|**sr.str.extract(pat)**|sr의 각 원소에서 패턴에 맞는 첫 번째 문자열을 데이터프레임으로 반환합니다. 캡처링 그룹을 추출하므로 반드시 소괄호를 사용해야 합니다.|
|**sr.str.extractall(pat)**|sr의 각 원소에서 패턴에 맞는 모든 문자열을 데이터프레임으로 반환합니다. 캡처링 그룹을 추출하므로 반드시 소괄호를 사용해야 합니다.|

## **정규 표현식**

 - 문자열 처리를 하기 위한 정규표현식을 설명한다.

 - 원하는 단어 , 원치않은 단어를 생략하고 출력하는데 주로 사용한다!

####  **한 글자 패턴 표현식 리스트**

|   |   |   |   |
|---|---|---|---|
|**구분**|**상세 내용**|**구분**|**상세 내용**|
|**'.'**|줄바꿈을 제외한 모든 문자(공백 포함)|'[a-z]'|알파벳 소문자(26개)|
|** '\\w'**|알파벳 대소문자, 숫자, 언더바, 한글|'[A-Z]'|알파벳 대문자(26개)|
|** '\\d'**|숫자|'[0-9]'|숫자(10개)|
|** '\\s'**|공백, '\r\n'(줄바꿈), '\t'(탭)|'[ㄱ-ㅎ]'|한글 단자음, 쌍자음, 겹자음(30개)|
|** '\\W'**|'\\w'의 반대|'[ㅏ-ㅣ]'|한글 단모음, 이중모음(21개)|
|** '\\D'**|'\\d'의 반대|'[가-힣]'|한글 완성형(11,172개)|
|** '\\S'**|'\\s'의 반대|'[一-龥]'|한자(20,902개)|
|** `'|'`**|문자열을 or로 지정|'[^가-힣]'|

####  **수량자, 위치 고정 , 그룹화 표현식 리스트**

|   |   |   |   |
|---|---|---|---|
|**구분**|**상세 내용**|**구분**|**상세 내용**|

|'{n,m}'|앞 패턴이 n~m번 연속 일치 ($n < m$)|** '\\n'**|n번째 캡처링 그룹 재탐색(역참조)|

## **조건에 맞는 행 선택하는 방법**

 - loc 인덱서의 행부분에 해당하는 조건 식을 넣어주면 된다.

 - **원하는 열에 원하는 행 출력을 위한 조건식을 넣어보자**

 - 조건을 넣는것도 좋지만 **시리즈의 비교 연산 매서드를 사용해서도 조회할수있다**

 - 조건에 맞는 행을 선택하기 위해서 **isin 매서드를 사용해서도 조회할수있다.**


#### **시리즈의 비교 연산 메서드**

|   |   |
|---|---|
|**구분**|**상세 내용**|
|**sr.gt(other = value)**|sr 원소가 value 초과면 True, 아니면 False인 불리언 시리즈를 반환합니다.  <br>gt는 'greater than'이므로 (sr > value)를 실행한 결과와 같습니다.|
|**sr.ge(other = value)**|sr 원소가 value 이상이면 True, 아니면 False인 불리언 시리즈를 반환합니다.  <br>ge는 'greater than or equal to'이므로 (sr >= value)를 실행한 결과와 같습니다.|
|**sr.lt(other = value)**|sr 원소가 value 미만이면 True, 아니면 False인 불리언 시리즈를 반환합니다.  <br>lt는 'less than'이므로 (sr < value)를 실행한 결과와 같습니다.|
|**sr.le(other = value)**|sr 원소가 value 이하면 True, 아니면 False인 불리언 시리즈를 반환합니다.  <br>le는 'less than or equal to'이므로 (sr <= value)를 실행한 결과와 같습니다.|
|**sr.eq(other = value)**|sr 원소가 value 같으면 True, 아니면 False인 불리언 시리즈를 반환합니다.  <br>eq는 'equal to'이므로 (sr == value)를 실행한 결과와 같습니다.|
|**sr.ne(other = value)**|sr 원소가 value 다르면 True, 아니면 False인 불리언 시리즈를 반환합니다.  <br>ne는 'not equal to'이므로 (sr != value)를 실행한 결과와 같습니다.|

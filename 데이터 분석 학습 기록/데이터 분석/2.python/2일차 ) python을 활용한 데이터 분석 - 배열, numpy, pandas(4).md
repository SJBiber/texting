## **데이터 프레임 생성**

```
import pandas as pd
df = pd.DataFrame(data = {'A':[1,2,3], 'B':[6,7,8]})

```

## **데이터 프레임 인덱싱**

#### **1. iloc 인덱서를 사용한 인덱싱**

 - 위치 인덱스로 데이터를 추출함

 - iloc의 대괄호 안 왼쪽에는 행 인덱스 , 오른쪽에는 열 인덱스를 지정한 후 조회할수있음

 - 이때 **행 또는 열 인덱스를 단일로 지정하면 시리즈로 , 리스트 형식으로 행 , 열 데이터를 지정하면 데이터 프레임으로 출력한다!**

```
#위치 인덱스인  0번 행의 열을 조회하고 싶어
df.iloc[0, :]     # 시리즈로 반환

# 출력
A    1
B    6

 # 위치 인덱스인 0~1번 인덱스 행을 조회
 df.iloc[0:2, :]    # 데이터 프레임으로 반환
 
 # 출력
 	A	B
0	1	6
1	2	7

# 위치 인덱스 1번과 0번을 출력하고 싶어
df.iloc[[1,0], : ]    # 데이터 프레임으로 반환

 # 출력
 	A	B
1	2	7
0	1	6

# 0번 위치 인덱스 열을 출력하고 싶어
df.iloc[: , 0]   # 시리즈로 반환

# 출력
0    1 
1    2
2    3
Name: A, dtype: int64

# df의 1번과 0번 인덱스 열을 출력하고 싶어
df.iloc[ :,[1,0]]     # 데이터 프레임 형태로 반환

# 출력
	B	A
0	6	1
1	7	2
2	8	3
```

#### **2. loc 인덱서를 사용한 인덱싱**

 - 행 이름 인덱스 데이터로 데이터를 추출함

 - iloc의 대괄호 안 왼쪽에는 행 인덱스 , 오른쪽에는 열 인덱스를 지정한 후 조회할수있음

 - 이때 **행 또는 열 인덱스를 단일로 지정하면 시리즈로 , 리스트 형식으로 행 , 열 데이터를 지정하면 데이터 프레임으로 출력한다!**

```
# 행이름이 1인 행을 시리즈로 반환
df.loc[1,:]

# 출력
A    2
B    7
Name: 1, dtype: int64

# 행 이름이 1,2 인 행을 데이터 프레임으로 반환 
df.loc[1:2,:]

# 출력
	A	B
1	2	7
2	3	8

# 행이름이 2, 1인 행을 데이터 프레임으로 변환
df.loc[[2,1],:]

# 출력
	A	B
2	3	8
1	2	7


# 열 이름이 A 인 열을 시리즈로 반환
df.loc[:,'A']

# 출력
0    1
1    2
2    3
Name: A, dtype: int64

# 열이름이 B,A인 열을 데이터 프레임으로 반환
df.loc[:,['B','A']]

# 출력
	B	A
0	6	1
1	7	2
2	8	3
```

## **데이터 프레임 행 / 열 추가 , 변경 및 삭제**

 - 데이터 프레임에 없는 행 또는 열 이름을 추가하고 , 리스트 할당

```
# 인덱스가 4인 행이름을 추가 및 리스트 할당
df.loc[4,:] = [11,12]

# 출력
	A	B
0	1.0	6.0
1	2.0	7.0
2	3.0	8.0
4	11.0	12.0


# 데이터 프레임에 없는 열이름을 추가하고 , 리스트를 할당
# 생성할 열에 행의 개수를 맞춰서 리스트를 할당해야한다! 아니면 오류 발생!!
df.loc[:,'D'] = ['a','b','c','d']

# 출력
	A	B	D
0	1.0	6.0	a
1	2.0	7.0	b
2	3.0	8.0	c
4	11.0	12.0	d
```

- 데이터 프레임에서 변경하려는 셀 값에 다른 값을 할당

```
# 데이터 프레임에서 변경하려는 셀값에 다른 값을 할당
df.loc[4,'D'] = 'F'

#출력
	A	B	D
0	1.0	6.0	a
1	2.0	7.0	b
2	3.0	8.0	c
4	11.0	12.0	F
```

- 데이터 프레임의 drop 매서드는 행이름으로 삭제하고 , 열 이름으로 열을 삭제

```
# 데이터 프레임의 drop 메서드는 행이름으로 행 삭제 , 열 이름으로 열 삭제
# index는 지우고 싶은 행 이름 , columns는 지우고 싶은 열 이름
# 둘중 하나만 기입해도 가능
df.drop(index = 4 , columns = 'D')  # 행이름이 4인 , 열이름이 D인 프레임 삭제

# 출력
	A	B
0	1.0	6.0
1	2.0	7.0
2	3.0	8.0
```

## **인덱싱 방법에 따른 결과 비교**

 **- 리스트 , 배열 , 시리즈 , 데이터프레임간 인덱싱 방법에 따른 결과를 다음 표와 같이 정리겠음**

|         |     |     |     |        |
| ------- | --- | --- | --- | ------ |
| 구분      | 리스트 | 배열  | 시리즈 | 데이터프레임 |
| 인덱싱     | 원소  | 원소  | 원소  | 시리즈    |
| 슬라이싱    | 리스트 | 배열  | 시리즈 | 데이터프레임 |
| 펜시 인덱싱  | X   | 배열  | 시리즈 | 데이터프레임 |
| 불리언 인덱싱 | X   | 배열  | 시리즈 | 데이터프레임 |

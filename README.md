# 📊 데이터 분석 및 엔지니어링 학습 저장소

데이터 분석 부트캠프 교육생이자 Git 개발자로서, 데이터의 수집부터 분석, 그리고 시각화까지의 전 과정을 기록하고 관리하는 공간입니다.

---

## 📂 주요 학습 및 프로젝트 활동

### 0️⃣ 실전 프로젝트 (`데이터 분석/0.프로젝트`)
- **데이터 문해력 및 기획**: 데이터 기반의 문제 해결 전략 수립 및 디지털 전환(DX) 기획
- **프로젝트 파이프라인**: 아이디어 구체화, 기획안 선정 및 실제 데이터 수집 프로세스 구축

### 1️⃣ 통계 기초 (`데이터 분석/1.통계`)
- **데이터 분석의 이론적 토대**: 분석에 필수적인 기본 통계 개념 및 확률 분포 학습
- **분석 용어 정립**: 혼동하기 쉬운 데이터 분석 용어들의 명확한 정의 및 체계화

### 2️⃣ Python 데이터 과학 (`데이터 분석/2.python`)
- **데이터 핸들링**: NumPy를 이용한 수치 연산 및 Pandas를 활용한 고수준 데이터 조작
- **전처리 기술**: 결측값 처리, 시계열 데이터 분석, `apply` 함수를 이용한 데이터 변환 효율화
- **데이터 시각화**: 시나리오 기반의 Matplotlib/Seaborn 차트 생성 및 인사이트 도출

### 3️⃣ 데이터 엔지니어링 (`데이터 엔지니어링`)
- **인프라 및 DB**: PostgreSQL, Supabase 연동 및 SQL을 통한 효율적인 데이터 관리
- **업무 자동화 & AI**: 데이터 수집 자동화 및 RAG, 벡터 DB 등 최신 AI 기술 기반의 엔지니어링 실습
